# Faisal's Portfolio

A modern, responsive portfolio website showcasing the professional profile, experience, flagship products, and technical expertise of Faishal Khan, a Multi-Region IT Infrastructure Lead.

## Features

- **Interactive Profile Modal**: Detailed bio and professional information
- **Experience Timeline**: Comprehensive work history and achievements
- **Flagship Products Showcase**: Highlighting key projects and solutions
- **Tech Stack Grid**: Visual representation of technical skills and tools
- **Contact Integration**: Direct links for WhatsApp, phone, and email communication
- **Responsive Design**: Optimized for desktop and mobile devices
- **Dark Theme**: Sleek, professional aesthetic with blue accent colors

## Tech Stack

- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom gradients and animations
- **Icons**: Lucide React
- **Deployment**: Private hosting (not publicly accessible)

## Project Structure

```
src/
├── components/          # Reusable UI components
│   ├── Experience.tsx   # Work experience section
│   ├── Footer.tsx       # Contact footer
│   ├── Hero.tsx         # Main landing section
│   ├── Navigation.tsx   # Top navigation bar
│   ├── Products.tsx     # Flagship products grid
│   ├── ProfileModal.tsx # Detailed profile popup
│   └── TechStack.tsx    # Technical skills display
├── data/                # Static data files
│   ├── experience.ts    # Work experience data
│   ├── products.tsx     # Product information
│   └── profile.ts       # Personal profile data
└── utils/               # Utility functions
    └── helpers.ts       # Helper functions for contact links
```
